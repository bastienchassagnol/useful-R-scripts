# Start a R package

- usethis::use_github_links() (to automatically add the URL associated with the package in the Gh description)
- `usethis::use_github_action()` options 1 and 2 to add core GitHub actions
- usethis::use_pkgdown_github_pages() to start documenting a GitHub repository
- precommit::use_precommit() to use pre-commits to automatically check the content of the folder before running out of GH credits
- Add coding instructions, namely `npx mdskills install PatrickJS/cursor-r`
